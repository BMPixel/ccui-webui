## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Open http://localhost:5173

## Technology Stack

- React 18 + TypeScript
- Vite build tool
- Tailwind CSS + shadcn/ui
- Zustand state management
- TanStack Query for API
- React Router for navigation
